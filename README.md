# 🧠 心智核磁共振图谱 (Mind-MRI Atlas)

示例站点：[mri.hocassian.com](点我查看)

**「心智核磁共振图谱」** 是一个基于 Web 的交互式心理病理学可视化工具。本项目致力于打破精神医学（冷峻的神经生物学数据）与心理治疗（深邃的人文精神动力学）之间的壁垒，构建一个多维度的映射模型。

它不仅展示了各种表象心理问题背后的神经系统激活规律，更深入剖析了主体在症状中隐秘的「剩余快感（Surplus Jouissance）」，并提供了对应的治疗与疏导策略。

## 🌌 核心设计理念

传统的 DSM/ICD 诊断标准往往基于症状的表面聚类。本项目采用了更为底层和跨维度的双轨框架：

1. ​**科学轨：NIMH RDoC 框架**​（研究领域标准）。将心理异常拆解为 6 个基础神经系统的异常激活水平（负向效价、正向效价、认知系统、社会过程、唤起调节、感知运动）。
2. ​**人文轨：当代精神动力与认知集成**​。融合了拉康（Lacan）、弗洛伊德（Freud）、比昂（Bion的阿尔法功能）、科胡特（Kohut的自体心理学）、福纳吉（Fonagy的心智化），以及范德科尔克（van der Kolk）的躯体创伤理论。

## ✨ 核心功能与模块

图谱为每一种心理状态/障碍提供了 5 个维度的深度切片：

* 📊 ​**RDoC 交互式雷达图**​：直观可视化 6 大神经系统的异常偏离度与激活状态。
* 🩺 ​**临床表象**​：精准描述该状态在现实检验和日常生活中的具象表现。
* 🧬 ​**RDoC 生物学机理**​：从多巴胺回路、HPA轴、前额叶-杏仁核连接等神经科学视角解释“大脑发生了什么”。
* 🎭 ​**成因分析（精神动力与认知）**​：深潜至潜意识层面，解析\*\*“隐秘的剩余快感”\*\*、大他者凝视、不耐受不确定性、心智化断崖式跌落等核心底层逻辑。
* 🌱 ​**疏导与治疗策略**​：结合 EBR 躯体能量转化、暴露疗法（ERP）、辩证行为疗法（DBT）、躯体体验（SE）等，提供临床介入方向。
* 🎬 ​**文艺镜像与文本投射**​：以影史经典电影（如《海边的曼彻斯特》《梦之安魂曲》《心灵捕手》等）作为绝佳的精神分析样本，辅助理解理论在活生生的人性中的映射。

## 🧩 已收录的心智图谱

目前版本完整刻画了 10 种核心心智光谱：

* **抑郁图谱** (Major Depressive) - *自恋性丧失与无名之丧*
* **焦虑图谱** (Anxiety Spectrum) - *全能控制感与世俗凝视*
* **强迫图谱** (Obsessive-Compulsive) - *不确定性不耐受与责任膨胀*
* **边缘型图谱** (Borderline Personality) - *心智化的断崖式跌落*
* **自恋型图谱** (Narcissistic Personality) - *全能自恋与大他者的镜像*
* **回避型图谱** (Avoidant Personality) - *羞愧感与主动丧失的快感*
* **双相图谱** (Bipolar Spectrum) - *全能防御的绝望狂欢*
* **成瘾图谱** (Addiction Spectrum) - *自我用药假说与短路大他者*
* **思觉失调图谱** (Schizophrenia Spectrum) - *阿尔法功能的失败*
* **PTSD图谱** (Trauma & PTSD) - *强迫性重复与躯体闪回*

## 🚀 快速开始

本项目为纯前端单文件（Single-File）架构，开箱即用，无需复杂的构建过程。

1. **克隆仓库**
2. **本地运行**直接在浏览器中双击打开 `rdoc-canvas.html` 文件即可体验完整的交互功能。

> ​**技术栈依赖**​：项目通过 CDN 引入了 [Tailwind CSS](https://www.google.com/search?q=https://tailwindcss.com/) (用于现代 UI 渲染) 和 [Chart.js](https://www.google.com/search?q=https://www.chartjs.org/) (用于绘制平滑的交互式雷达图)，以及 Google Noto 字体库以保证学术排版质感。请确保在有网络连接的环境下打开。
